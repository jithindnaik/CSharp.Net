# CSharp.Net

Grade App

Expectation -
1. User will add a mark in the command prompt
2. Grade will display based on the mark with Pass / Fail. Example: Mark: 50, Result: Fail

Technicals -
Mark < 50   ->  Fail
Mark > 50 && < 55    ->  D Grade
Mark > 55 && < 65    ->  C Grade
Mark > 65 && < 75    ->  B Grade
Mark > 75 && < 90    ->  A Grade
Mark > 90 && < 100  ->  A+ Grade
